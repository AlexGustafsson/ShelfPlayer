#  ShelfPlayer

A iOS 17 native Audiobook & Podcast player [audiobookshelf](https://www.audiobookshelf.org)

## Features

- Online & Offline Playback
- View audiobooks, podcasts, episodes, series & authors
- Search your libraries
- Sleep timer
